# Vidisha Educational Society

The official web platform for **Vidisha Educational Society** and **Vidisha Junior College**. This application is designed to showcase our academic excellence, modern campus facilities, and our specialized coaching programs for MPC and BiPC streams.

##  About Vidisha

Founded in 2023 in Vijayawada, Vidisha Educational Society is dedicated to empowering the next generation of engineers and doctors. Our institution provides a specialized learning environment that integrates intermediate board education with intensive IIT-JEE and NEET coaching.

## Key Features

- **Academic Streams**: Detailed insights into our MPC (Engineering) and BiPC (Medical) programs.
- **Interactive Campus Gallery**: A visual tour of our state-of-the-art laboratories, smart classrooms, and campus life across various branches.
- **Direct Admissions Portal**: Streamlined contact and registration forms for prospective students and parents.
- **Dynamic Branch Locator**: Easy access to information for all our branches, including Vijayawada (Novotel, Benz Circle, Edupugallu, Bharathi Nagar), Visakhapatnam, Guntur, Hyderabad, and Kakinada.
- **Responsive Design**: A premium, mobile-first experience optimized for all devices.

## Technology Stack

- **Frontend**: React 19
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Build Tool**: Vite

##  Getting Started

To run this project locally, follow these steps:

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/harshadeepkota3-star/vidisha-final.git
    ```

2.  **Navigate to the project directory**:
    ```bash
    cd "Vidisha Educational Society"
    ```

3.  **Install dependencies**:
    ```bash
    npm install
    ```

4.  **Start the development server**:
    ```bash
    npm run dev
    ```

The application will be available at `http://localhost:5173` (or the port specified in your terminal).
